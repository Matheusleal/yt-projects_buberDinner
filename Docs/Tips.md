# Tips

## User-secrets

### Init user-secrets
```bash
    dotnet user-secrets init --project <path-to-exe-project>
```

### Add user-secrets
```bash
    dotnet user-secrets set --project <path-to-exe-project> <"pathInJson:Settings">
```

### List user-secrets
```bash
    dotnet user-secrets list --project <path-to-exe-project>
```
